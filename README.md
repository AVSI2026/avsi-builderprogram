# AV Solutions & Integration - Builder Partnership Program

A modern, responsive marketing website for AV Solutions & Integration's builder partnership program. Built with React, Vite, Tailwind CSS, and Lucide icons.

## Features

- **Responsive Design** - Mobile and desktop optimized layouts
- **Dark Theme** - Premium dark mode with gold accents (#c5a059)
- **Icon Integration** - Lucide React icons throughout
- **Interactive Pricing Tiers** - Three-tier service offerings (Connected Home, Smart Home, Luxury Living)
- **Service Workflow** - Visual workflow table showing 4 project phases
- **Professional Branding** - Consistent design system with corporate colors

## Project Structure

```
src/
├── App.jsx           # Main application component
├── index.css         # Tailwind CSS directives
└── main.jsx          # React entry point
```

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173/`

### Build for Production

Create an optimized production build:

```bash
npm run build
```

### Preview Production Build

Preview the production build locally:

```bash
npm run preview
```

## Technologies Used

- **React 19** - UI framework
- **Vite 7** - Build tool and dev server
- **Tailwind CSS 4** - Utility-first CSS framework
- **Lucide React** - Icon library
- **PostCSS** - CSS transformation tool
- **Autoprefixer** - CSS vendor prefixing

## Project Features

### Sections

1. **Header/Hero** - Branded logo and value proposition

3. **Technical Scope** - Six service domain icons and labels
4. **Technology Tiers** - Three service packages with features and pricing

- Primary: `#0a0a0a` (Black)
